    人工智能/智能科学与技术评教

    老师排名不分先后!

    评教样例源码：

    1. <your id>@<comment time> - <your comment> - ...

    请注意：

    • 学期 > 课程 > 老师 > 年级
    • 低年级排序高于高年级：2020 级 > 2019 级
    • 同年级按时间顺序排：...@20220117 > ...@20220125

    第三学期

    离散数学

    刘丽珏

    2020 级

    1. heyzbw@20220218
    2. 珏珏子对离散的理解还是满深入的，认真听可以收获很多。
    3. 珏珏子声音有点轻，想听建议前排。
    4. 上课不是很在意人有没有来齐，没有点过名。
    5. 20级开始需要完成畅课上的练习，最好看完畅课课件，珏珏子中途不会提醒，但最后平时分她说会参考这个。

    2019 级

    1. NIUYI0511@20220130
    2. 珏珏子说话不着急，可能有些人觉得语速有点慢，声音也不是特别大，可以去抢前排，基本不会有点名提问。上课强烈建议好好听讲，听懂每一个知识点。
    3. 用的教材是她自己出的那本，西电出版社编的，还算比较好理解。
    4. 逻辑和图论几章是重点，但考试套路不深
    5. 每章结束会有几个题目要求完成，是平时成绩的组成。考试会出原题，只要认真听讲搞懂作业期末成绩可以比较高。

    数据结构

    刘丽珏

    2020 级

    1. heyzbw@20220218
    2. 课堂风格和离散数学类似。
    3. 20级没有课程设计，但每一章会设置几道力扣的题目，以小组为单位讨论并完成。
    4. 实验周也是按小组到实验室，以个人为单位检查，检查内容：力扣完成题数，任自选一道题讲解思路。
    5. 畅课上的题目也要按时完成，与离散数学同理。

    2019 级

    1. NIUYI0511@20220130
    2. 是的还是珏珏子，这学期她代两门。
    3. 这学期真正硬核的课程，尽可能听懂，课下多想想。
    4. 这门课有课程设计，老师出题，对于大二上学生有一定难度。小组合作，要做系统，部分题目涉及物联网RFID和人工智能图搜索的相关算法，也会有答辩。
    5. 考试不容易，要好好复习，但最后给分其实还可以。
    6. 课设和答辩是重点，珏珏子很在乎这一点，答辩耳目一新，课设做的有创新点是加分项。

    4. 会课堂提问，让同学上去做题（为了让我们拿到课堂分），本身的问题不是太难，不会超纲
    5. 考试难度适中，不会太为难学生，会希望学生都及格
    6. 有实验，要写实验报告（手写）

    8. M-Chase@20220208

    脑与认知科学基础

    邹逸群

    2020 级

    1. heyzbw@20220218
    2. 听说智能科学与技术改为人工智能就不上这门课了。
    3. 上课讲金庸时，不要一笑而过，可能会考。
    4. 开卷考试，考的时候满有信心，成绩出来就满纳闷，成绩20级普遍偏低。
    5. 20级课设是实现二分类，小组合作，和课程毫不相干，但挺有用的。

    2019 级

    1. NIUYI0511@20220130
    2. 教材用北邮的那本，王志良主编
    3. 金庸小说狂热爱好者，表情包收藏大师，会在PPT上插大量例子和表情包
    4. 课的内容不容易听懂，涉及脑科学，心理学，机器学习，人机交互等等，因此大家容易摆烂
    5. 考试开卷，会有一两个题出自课上他举的例子，（19届考试出了神雕侠侣和射雕英雄传的题目，左右互搏巴拉巴拉，当时给看傻了）
    6. WARNING：从19级开始，这个课开始有课程设计，题目对于大二上学生很难，是机器学习的内容，19级做的是使用国际第三届脑机接口比赛（BCI-Ⅲ）数据做脑电信号分类，主流是用LDA和SVM
    7. 会有课设答辩，会把你问住，但也不会太为难，阿弥陀佛。。。

    高琰

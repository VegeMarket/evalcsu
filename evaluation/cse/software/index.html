    软件工程评教

    老师排名不分先后!

    评教样例源码：

    1. <your id>@<comment time> - <your comment> - ... 请注意： - 学期 > 课程 > 老师 > 年级 - 低年级排序高于高年级：2020 级 > 2019 级 - 同年级按时间顺序排：...@20220117 > ...@20220125

    第一学期

    计算机程序设计基础（C++）

    谭长庚

    2019级

    1. EZChen@20220218
    2. 老师相当负责任，上课很认真
    3. C++ 属于偏难的语言，建议自己多用功
    4. 实验量极大，需要自己课下完成

    信息系统基础SSD1

    杨柳

    2019级

    1. EZ4Chen@20220218
    2. 新生的当头一棒，备课和上课相当随意，实验基本由研究生指导
    3. 考试比较离奇，和上课内容基本无关，且全是英文题
    4. 考试前会发资料和重点

    高等数学A2（一）

    张力

    2019级

    1. EZ4Chen@20220218
    2. 老师说话带有方言且模糊，且教室的扩音设备基本无效。建议第一排入座（重要
    3. 和善且负责，喜欢同学提问题
    4. 很少点名，但是期末成绩必须在 45 分以上才能及格

    陈学工

    宋铁

    2019级

    创新创业导论

    杨俊

    2019级

    1. EZ4Chen@20220218
    2. 鸡汤混杂，脱离实践
    3. 会在国庆前，考试前点名，会扣分。
    4. 有期中考试，不会提前通知。千万不能不来

    大型数据库技术

    邓磊

    2019级 1. EZ4Chen@20220218 - 快逃 - Oracle的配置请不要按照他的指导编写，使用docker快速部署才是正解 - 考试前会划重点，都会考，没有遗漏

    云计算及应用

    邓磊

    2019级 1. EZ4Chen@20220218 - 快逃 - 考试前会划重点，都会考，没有遗漏 - 建议使用 Vmware 构建 Linux 集群，不建议购置云服务器进行试验 - 实验指导书的最早编写时间应该是 2012 年前后，基本没有注释和说明，只有配置的内容和程序样例 - 相关技术: hadoop
